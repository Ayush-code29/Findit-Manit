# FindIt MANIT

FindIt MANIT is a full-stack Lost & Found web application built exclusively for MANIT students. The platform helps students report lost items, post found items, submit ownership claims, and reconnect with their belongings through a secure and user-friendly system.

## Features

### Authentication
- Secure user authentication using Clerk
- Protected routes and user-specific dashboards

### Lost & Found Management
- Report Lost Items
- Report Found Items
- Upload item images
- Categorize items
- Location and contact details

### Claim System
- Ownership verification through claim requests
- Claim approval and rejection workflow
- Real-time notifications for item owners

### User Dashboard
- View personal reports
- Manage lost and found listings
- Delete reports
- Track claim requests

### Notifications
- Claim request alerts
- Approval and rejection notifications

### Cloud Storage
- Image uploads using Cloudinary
- Optimized image compression before upload

### Responsive Design
- Mobile-friendly UI
- Modern dashboard design
- Dark Mode support
